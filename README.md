## 🎯 Kawata/Aeris <a name="kawata-aeris"></a>
_Originally developed by **Panini Céleste**. Currently maintained by **TheFantasticLoki**, with contributions from Maple Syrup and Chewy/Pythr._

**Download:** https://storage.kawata.pw/get/osu!Kawata.zip *(public beta)*  
**Virus analysis:** https://www.hybrid-analysis.com/sample/3a08fea940bb7028b08b0a6688cae86344af3fc5ea2340ff03a29d95be090614

> 💡 Add the install folder as an antivirus exclusion to prevent issues.

**Cheats:** Aim Correction (improved Skooter AC, optional Tap-on-Correct for non-RX, relative range support) · Relax Hack (SkooterRX) · Timewarp (Rate & Multiplier) · Automatic CS Changer · AR Changer · HD Remover · FL Remover

**Beta:** Maple Aim Assist (V1 + V2 submitting; V3 and L-V1 not submitting) · Dedicated input thread (aim only — 1/2/4/8/16K FPS options)

## 🎯 osu!rx <a name="osu-rx"></a>
_Free hack by **mrflashstudio**, updated by **Sasuke**._

**Download:** https://www.mpgh.net/forum/showthread.php?t=1538659

**Setup:**
1. osu!rx does **not** run on the latest osu!. Download [b20220424](https://osekai.net/snapshots/versions/b20220424/b20220424.zip).
2. Apply [this patched exe](https://cdn.discordapp.com/attachments/598976475579809860/1082594775988981760/osu.exe) (raises TLS version).
3. Create a `_STAGING` file in your osu! folder to block auto-updates.
